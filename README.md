# Spring-Playground
Inflearn 강의를 통해 Spring의 구조 이해와 JDBC, JPA, QueryDSL 순으로 공부하여,
웹 프레임워크에 대한 이해와 DB ORM에 대한 이해를 증강시키기 위해 만든 레포지토리입니다.

공부하는 과정에서 객체지향에 대한 이해도와 코드 Standard(Upper LowerCase)를 지키기 위해 노력하는 방향을 갖고 있습니다.

## 1. Core - 스프링 핵심 원리 강좌
---
JAVA 객체지향의 특성부터 DI, OCP 등 SOLID 객체지향 원칙을 지키면서 프로그래밍하는 방법론에 대해 공부합니다.
도메인을 개발하며 Interface와 Impl(구현체 추상체)를 분리하면서 Solid 원칙을 지키는 방향으로 프로그래밍 합니다.
이후 SOLID 원칙을 지킬 경우 새로운 정책을 최대한 코드 변화 없이 웹 개발에 적용하는 모습을 학습하며,
SOLID 원칙을 지켜야 하는 이유에 대해 체감합니다.

AppConfig Refactoring을 통해 코드 가독성을 높이는 방법에 대해서 공부했습니다. (DI관련)

초반에는 Spring Bean 없이 Native 자바로 테스트 코드를 짜고, 
후반부에 가서는 Spring Bean과 Spring Container를 이용하여 편리하게 DI 할 수 있는 방법을 공부합니다.

싱글톤 패턴의 공부를 통해 Java Design Pattern에 입문하였습니다.
객체를 처음 한개만 생성하고 이후에는 객체생성을 제한하는 방법으로 여러 객체를 중복 생성하지 않도록 하는 패턴입니다.

이외에도 자, 수동 인식 Spring Bean(Component Configuration .. etc)에 대해 배우면서 실무에서 코드가 꼬이지 않고 최적의 방법으로 가독성 높은 코드를 짜는
방법론에 대해서 공부합니다.
